# ReconForge

A modular Python-based reconnaissance framework built with Python for penetration testing, bug bounty, and security assessments.

---

## Features

- DNS Lookup
- WHOIS Lookup
- HTTP Header Analysis
- Port Scanner
- Banner Grabbing
- SSL Information
- Security Headers Analysis
- Technology Detection
- Subdomain Enumeration
- Directory Enumeration *(Coming Soon)*

---

## Installation

### Clone the repository

```bash
git clone https://github.com/Amr-Amen/ReconForge.git
cd ReconForge
```

### Create a virtual environment

```bash
python3 -m venv venv
```

### Activate it

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Exit the virtual environment

```bash
deactivate
```

### Install ReconForge

```bash
chmod +x install.sh
./install.sh
```

---

## Usage

Run ReconForge from anywhere:

```bash
reconforge
```
